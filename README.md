# jsnippets - make status updating easier

## run

- make jsnippets pgsql user and db it is owner of
- use env vars to set PG_HOST, PG_PORT, PG_DB, PG_USER, PG_PASS
- execute: java -jar path/to/jsnippets.jar

## run requirements
- jre 17
- postgresql
- mail server (optional)

## build requirements

- jdk 17
- maven 3
